Chapter 4: Understanding Population of Populations

What are Metapopulations?

A metapopulation is a "population of populations" - a group of several local populations that are linked by immigration and emigration.

Key Conceptual Shifts

Measurement Shift: Size → Persistence

Instead of tracking population size (N), we focus on whether populations persist (1) or go extinct (0). We measure the fraction of sites occupied (f).

Spatial Scale Shift: Local → Regional

We study equilibrium at the landscape level rather than focusing on individual populations. Local extinctions are normal, but regional persistence is the goal.

Fundamental Metapopulation Equation
$\frac{df}{dt} = I - E$
Rate of change in fraction of occupied sites equals immigration minus extinction

Extinction Risk: Local vs Regional

Local Extinction Probability (pe): 0.7
Number of Patches (x): 5
Regional Persistence Probability
$P_x = 1 - (p_e)^x$
Multiple patches "spread the risk" of extinction

Four Metapopulation Models

Island-Mainland Model

Assumptions: Constant colonization from external source (mainland), independent extinctions

$\frac{df}{dt} = p_i(1-f) - p_e f$ $\hat{f} = \frac{p_i}{p_i + p_e}$
Equilibrium fraction of occupied sites

Key Insight: Metapopulation always persists (f̂ > 0) due to external propagule rain, even if pe >> pi

Internal Colonization Model

Assumptions: Colonization only from occupied patches, no external source

$\frac{df}{dt} = if(1-f) - p_e f$ $\text{Equilibrium: } \hat{f} = 1 - \frac{p_e}{i}$

Key Insight: Metapopulation persists only if i > pe. If condition not met, regional extinction occurs!

Rescue Effect Model

Assumptions: External colonization + immigration reduces extinction probability

$\frac{df}{dt} = p_i(1-f) - ef(1-f)$ $\text{Equilibrium: } \hat{f} = \frac{p_i}{e}$

Key Insight: If e < pi, landscape becomes saturated (f̂ = 1). Immigration "rescues" declining populations!

Combined Model

Assumptions: Internal colonization + rescue effect (fully closed system)

$\frac{df}{dt} = if(1-f) - ef(1-f)$ $\text{Outcome depends on i vs e relationship}$

Key Insight: If i > e → saturation; if e > i → extinction; if i = e → neutral equilibrium

Question 1: Island-Mainland Model

An ant lion metapopulation has pi = 0.2 and pe = 0.4. What fraction of islands will be occupied at equilibrium?

Correct Answer: B) 0.33
Using the island-mainland equilibrium formula: f̂ = pi/(pi + pe) = 0.2/(0.2 + 0.4) = 0.33

Question 2: Internal Colonization

A metapopulation with internal colonization has i = 0.3 and pe = 0.5. What will happen to this metapopulation?

Correct Answer: B) Go regionally extinct
Since i (0.3) < pe (0.5), the metapopulation cannot persist. The equilibrium f̂ = 1 - pe/i = 1 - 0.5/0.3 < 0, indicating extinction.

Question 3: Risk Spreading

Why do multiple patches reduce extinction risk even when each patch has high local extinction probability?

Correct Answer: B) Regional persistence probability increases with patch number
The probability of regional persistence Px = 1 - (pe)^x increases rapidly as x increases, even with high pe. This is the "spreading the risk" principle.

Question 4: Rescue Effect

Which statement best describes the rescue effect in metapopulations?

Correct Answer: B) Immigration reduces local extinction probability
The rescue effect occurs when immigrants boost local population sizes, reducing the probability of extinction due to demographic and environmental stochasticity.

Question 5: Model Assumptions

Which assumption is NOT typically made in basic metapopulation models?

Correct Answer: C) Spatial arrangement of patches matters
Basic metapopulation models assume NO spatial structure - colonization probability depends only on overall f, not on spatial arrangement of occupied patches.

Question 6: Conservation Application

For the Bay Checkerspot butterfly, why is protecting the Morgan Hill population critical?

Correct Answer: B) It serves as the mainland source for recolonization
Morgan Hill is the large, persistent population that provides colonists for empty patches after local extinctions, making it essential for metapopulation persistence.
